a 494 byte (gzipped and minified) css starter

This is the result of me playing css "code golf" with Skeleton. As such, this framework strives to maintain all of the features that Skeleton and other similar frameworks (Milligram, Min... etc) offer.

I would love help in making this smaller. The only rules are that newer versions look similar to the current version. Optimizing this for the gzip algorithim has been an interesting, but fun challenge.

Any other thoughts? Head to the Github link below and make an issue! If you like this idea, don't forget to star!
