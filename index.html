world's smallest css framework* (465 bytes gzipped and minified)

This is the result of me playing css "code golf" with Skeleton. As such, this framework strives to maintain all of the features that Skeleton and other similar frameworks (Milligram, Min... etc) offer.

Any other thoughts? Head to the Github link below and make an issue! If you like this idea, don't forget to star!

Github

* Depends on how you define framework. For me, a framework needs to have default styles (buttons, typography, forms), and a grid system.

This is a paragraph. The font family is Nunito. Some bold text. Italics also looks nice. You can underline things you feel are important. Lit has styles that override the default heading styles, including margin.

    You can <u>underline</u> things you feel are important.

Menu

While lit doesn't come with a menu component, you can use grids to try and make a menu. Keep in mind that the grid wasn't designed for this, so you may experience weird spacing issues. I reccomend adding some custom css to do spacing.

Brand News About Docs Login 

    <div class="row">
        <b class="1 col">Brand</b>
        <a class="1 col" href="#">News</a>
        <a class="1 col" href="#">About</a>
        <a class="1 col" href="#">Docs</a>
        <a class="1 col" href="#">Login</a>
    </div>
